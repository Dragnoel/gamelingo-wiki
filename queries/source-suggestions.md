## Finding: game system identified

The review terms (突撃段階, 射撃段階, 白兵戦段階, 移動段階, 士気, 耐久力, 敏捷性, 加護) are
confirmed Japanese terminology from **Warhammer Fantasy Battle 8th Edition** (Games Workshop, 2010),
specifically the Japanese community translation conventions used in the Japanese box set release and
associated free PDF army books. Several terms also appear in **Warhammer: The Old World** (2024),
the successor game that uses the same phase and stat vocabulary.

The okinawahammer.livedoor.blog community site uses 突撃段階, 射撃段階, 白兵戦段階, 士気,
and 耐久力 directly in rules tutorials for WFB 8th edition.

- **Author/Publisher:** Games Workshop
- **Date:** 2010 (box set); army book PDFs released subsequently
- **URL:** Available from Games Workshop Japan; referenced at

- **Relevance:** Primary Japanese-language source for all 36 review terms. The WFB 8th edition
  Japanese box set established the Japanese terminology for phases (段階), stats, and mechanics
  used across the Japanese miniature wargame community. Japanese army book PDFs were made
  available as free downloads for box set owners.
- **Availability:** Freely available (PDF) — check Games Workshop Japan support or
  the WFB Japanese community for current download links.
- **Suggested raw/ subdirectory:** `raw/manuals/`

## Source 2 — Warhammer: The Old World Core Rulebook v1.0 (EN, free PDF)

- **Title:** Warhammer: The Old World — The Rules (v1.0)
- **Author/Publisher:** Games Workshop
- **Date:** January 2024
- **URL:** https://www.warhammer-community.com/wp-content/uploads/2024/01/EpbbHMiNWHnmBxvP.pdf
- **Relevance:** The Old World is the direct successor to WFB and uses the same stat and phase
  vocabulary. This PDF defines Aegis (加護) Save, Resilience (耐久力), Discipline (士気), and
  Penetration (貫通力) as named mechanics. Essential for cross-referencing how WFB 8th edition
  Japanese terms map to the current English canonical names.
- **Availability:** Freely available (direct PDF link from official Games Workshop Community site).
- **Suggested raw/ subdirectory:** `raw/manuals/`
